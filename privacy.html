Politique de confidentialité - Ant'icipate (FR)

Ant’icipate collecte uniquement les données nécessaires à l’identification des espèces de fourmis.

Données collectées :

  • Photos des fourmis envoyées par l’utilisateur
  • Informations facultatives : taille estimée, lieu d’observation, date

Utilisation des données :

Ces données sont utilisées uniquement pour fournir la fonctionnalité principale de l’application : identification des espèces de fourmis via un modèle d’intelligence artificielle.

Partage des données :

Aucune donnée n’est partagée avec des tiers. Les images et informations fournies restent confidentielles.

Sécurité des données :

Toutes les données transmises sont protégées via HTTPS. Elles ne sont pas stockées de façon permanente.

Suppression des données :

Les données liées aux prédictions (historique) sont stockées localement sur votre appareil. Vous pouvez les supprimer à tout moment via l’option “Supprimer” dans l’historique des prédictions de l’application. Si vous désinstallez l’application, toutes les données locales sont automatiquement effacées.

Contact :

Pour toute question concernant la confidentialité, contactez : ton-email@example.com

Privacy Policy - Ant'icipate (EN)

Ant’icipate only collects data necessary to identify ant species.

Data collected:

  • Photos of ants uploaded by the user
  • Optional information: estimated size, location, date

Purpose of data usage:

Data is used only to provide the main app functionality: identifying ant species via an AI model.

Data sharing:

No data is shared with third parties. All submitted information remains confidential.

Data security:

All data is transmitted via HTTPS and not stored permanently.

Data deletion:

Prediction data (history) is stored locally on your device. You can delete it at any time using the “Clear” option in the app’s prediction history. Uninstalling the app will automatically erase all local data.

Contact:

For privacy questions, contact: your-email@example.com
